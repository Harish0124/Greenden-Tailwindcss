# Greenden-Tailwindcss
## DESCRIPTION

Greenden is a responsive plant shopping website built using **HTML5** and **Tailwind CSS**. It features a clean and modern design with sections such as a navigation bar, hero banner, product collection, customer reviews, newsletter subscription, and footer.

## FEATURES

- Responsive design
- Navigation bar
- Hero section
- About section
- Product collection
- Customer reviews
- Newsletter subscription
- Contact page
- Modern UI using Tailwind CSS

## TECHNOLOGIES USED

- HTML5
- Tailwind CSS

## PROJECT STRUCTURE

```
Greenden-TailwindCSS/
│── index.html
│── product.html
│── contact.html
│── input.css
│── output.css
│── images/
└── README.md
```

## HOW TO RUN

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Greenden-TailwindCSS.git
   ```

2. Open the project folder.

3. Run the project using **Live Server** in Visual Studio Code.
